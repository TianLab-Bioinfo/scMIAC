This directory contains all experimental scripts for reproducing the results in the scMIAC paper, including competing method comparisons, ablation studies, and hyperparameter experiments.

## Directory Structure

```
scripts/
├── run_methods/           # Method execution scripts
│   ├── implementations/   # Implementation code for scMIAC and competing methods
│   ├── run_diagnal_methods/    # Execution scripts for diagonal integration methods
│   ├── run_vertical_methods/   # Execution scripts for vertical integration methods
│   └── utils/             # Utility functions and helper scripts
├── ablation/              # Ablation study experiments
├── hyperparameter/        # Hyperparameter tuning experiments
└── benchmark/             # Evaluation framework and benchmarking utilities
```
